# Emoji-Gen.ninja 功能调研文档

## 一、项目概述

### 1.1 项目简介
- **网站名称**: Emoji Generator (emoji-gen.ninja)
- **项目定位**: 免费的聊天工具表情符号生成服务
- **核心功能**: 通过指定文字和颜色生成可用于聊天工具（如 Slack）的 Reacji（反应表情）
- **目标用户**: Slack、Discord、Teams 等聊天工具的用户
- **技术栈**: Python、aiohttp、JavaScript、TypeScript、Vue、C++、Skia

### 1.2 产品特点
- 完全免费使用
- 操作简单快捷
- 支持自定义文字和颜色
- 生成高质量 PNG 图片
- 适配各类聊天平台的尺寸要求

---

## 二、核心功能分析

### 2.1 文字输入功能

#### 2.1.1 基本输入
- **输入行数**: 支持 **2 行文字** 输入
- **字符限制**: 建议每行 2-4 个字符（根据字体和大小自适应）
- **语言支持**:
  - 完整支持英文、数字、符号
  - 部分字体支持日文（注意：LinLibertine Bold 字体不支持日文）
  - 支持中文（需验证具体字体兼容性）

#### 2.1.2 文字对齐
- **左对齐** (Left Align)
- **居中对齐** (Center Align) - 推荐使用
- **右对齐** (Right Align)

#### 2.1.3 文字大小
- **自动调整**: 根据文字长度自动计算最佳大小（推荐）
- **手动调整**: 可固定文字大小
- **自适应逻辑**: 文字越长，字号自动缩小以适应画布

---

### 2.2 字体选择功能

#### 2.2.1 可用字体
根据调研，网站提供 **6 种字体选择**：
1. **默认字体** - 适合大多数场景
2. **LinLibertine Bold** - 粗体英文字体（不支持日文/中文）
3. **其他 4 种字体** - 需进一步确认具体名称

#### 2.2.2 字体特性
- 部分字体仅支持英文和数字
- 建议为不同语言提供字体兼容性提示
- 粗体字体更适合简短标语

---

### 2.3 颜色定制功能

#### 2.3.1 前景色（文字颜色）
- **颜色格式**: 支持 HEX 和 RGB
- **颜色选择器**: 图形化颜色选择工具
- **手动输入**: 可直接输入色值
- **透明度控制**: 可调整颜色强度/不透明度

#### 2.3.2 背景色
- **颜色格式**: 支持 HEX 和 RGB
- **透明背景**: 支持 **完全透明背景**（仅显示文字）
- **纯色背景**: 支持任意纯色背景
- **颜色强度**: 可调整背景色的透明度

#### 2.3.3 颜色预设
- 建议提供常用颜色预设方案
- 支持保存用户自定义配色

---

### 2.4 生成与导出功能

#### 2.4.1 生成格式
- **输出格式**: PNG 图片
- **图片尺寸**:
  - 标准尺寸: 128x128 像素（Slack 推荐）
  - 可能支持其他尺寸（需验证）
- **透明通道**: 支持 Alpha 通道（透明背景）

#### 2.4.2 实时预览
- 即时显示生成效果
- 所见即所得的编辑体验
- 参数调整后立即更新预览

#### 2.4.3 下载功能
- 一键下载生成的 PNG 文件
- 文件命名规则（需确认）
- 批量生成功能（待确认）

---

### 2.5 用户体验功能

#### 2.5.1 界面布局
- **简洁设计**: 单页面应用，所有功能集中展示
- **响应式设计**: 适配桌面和移动设备
- **操作流程**: 输入 → 定制 → 预览 → 下载

#### 2.5.2 辅助功能

## 三、技术架构分析

### 3.1 前端技术
- **框架**: Vue.js
- **语言**: JavaScript / TypeScript
- **界面**: 响应式单页应用

### 3.2 后端技术
- **语言**: Python
- **框架**: aiohttp（异步 HTTP 框架）
- **图形渲染**: Skia (C++)
  - 高性能 2D 图形库
  - 支持文字渲染、颜色处理
  - 跨平台支持

### 3.3 渲染流程
1. 接收前端参数（文字、颜色、字体、对齐等）
2. 使用 Skia 引擎渲染文字到画布
3. 应用颜色和样式
4. 生成 PNG 图片
5. 返回给前端/提供下载

## 七、关键发现与洞察

### 7.1 优势
1. **极简操作**: 3-5 步即可生成专业表情包
2. **高度定制**: 颜色、字体、对齐等全面可控
3. **技术先进**: 使用 Skia 保证渲染质量
4. **完全免费**: 无付费墙，无广告干扰

### 7.2 待优化点
1. **字体数量**: 仅 6 种，可扩展更多
2. **特效支持**: 无渐变、阴影等高级效果
3. **批量生成**: 似乎不支持批量创建
4. **模板系统**: 缺少预设模板快速选择

### 7.3 创新机会
1. **AI 建议**: 根据关键词自动推荐颜色和字体
2. **动画表情**: 支持生成 GIF 动画
3. **图案背景**: 支持纹理/图案背景
4. **尺寸预设**: 针对不同平台的尺寸预设
5. **表情管理**: 个人表情库管理系统

## 八、实现难点分析

### 8.1 技术难点
1. **跨平台字体渲染**: 确保不同系统显示一致
2. **中日韩字符支持**: CJK 字符的字体兼容性
3. **图片质量优化**: 小尺寸下的清晰度
4. **性能优化**: 实时渲染的响应速度

### 8.2 设计难点
1. **参数平衡**: 简洁 UI 与丰富功能的平衡
2. **默认值设定**: 新手友好的默认参数
3. **颜色选择器**: 直观高效的颜色工具
